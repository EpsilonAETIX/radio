# Radio (Lite & Advanced) - Version Control

This repository hosts the version control and changelog data for the **epsilon_radio** FiveM resource.

## Purpose

The primary purpose of this repository is to provide a centralized source for:
- **Update Checking:** The script checks this repository to see if a new version is available.
- **Changelogs:** Users can view the latest changes and updates directly within their server console upon resource start, eliminating the need to constantly check Discord or a website for patch notes.
